# Auth Service for JobiGo

This service implements authentication and user management for the JobiGo platform.

Features
- Django + DRF
- Custom User model (UUID PK)
- Roles: ADMIN, RECRUITER, SEEKER
- JWT Authentication (SimpleJWT)
- Registration, Login, Refresh, Password Reset, Email Verification
- Service / Repository pattern
- Dockerized

See README_MODULE.md for developer notes and running instructions.
